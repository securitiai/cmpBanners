IAB Multi-Level Banner - Opt-Out Text Test

Test Purpose

  1. This page is designed to test whether the opt-out text is correctly displayed in the IAB multi-level banner.
  2. There is a known issue where the opt-out tab gets removed from the UI under certain navigation conditions.

Configuration Details

Domain Group: iab-optout

Domain: https://iab-opt-out-issue-test.com

Steps to Reproduce the Issue

  1. Wait for the banner to load on page load (or click the "Manage Cookie Preferences" button below)
  2. Click on the "IAB Purposes" tab in the banner
  3. Navigate back to "Cookie Preferences" tab
  4. Expand the description for any cookie category
  5. Observe if the opt-out tab is still visible

Expected Issue

Problem: The opt-out tab gets removed from the UI after following the above steps.

Expected Behavior: The opt-out tab should remain visible and functional even after navigating between tabs and expanding descriptions.
